# はじめに

このウェブサイトは、僕 ([Kohei](https://github.com/axross)) がポーカーを勉強する中で知ったことをまとめたノートです。とりとめのない記述があったり、正確でない内容があったりします。ご容赦ください。

対象読者として、ポーカーの中級者で、中学数学の理解がある人を意識しています。僕がソフトウェアエンジニアなので、ソフトウェアエンジニア的な視点での書き方をしていることもあります。初めたばかりの人には色々と難しい内容も含まれているかもしれません。

また、このウェブサイトは全てノーリミットテキサスホールデムを前提としています。指定なく「ポーカー」と表記があれば、それはノーリミットテキサスホールデムのことを指していると思ってください。

フィードバック [yo@kohei.dev](yo@kohei.dev) までお願いします。
