# ハンドレンジを分類する

何らかのアクションを1回以上起こした後は、そのプレイヤーのハンドレンジに傾向を見ることができます。この傾向を分類することで、そのプレイでのプレイヤーの立ち位置を測ることができます。

すぐに思いつくのは、**プレイヤーのハンドレンジがキャップドであるか**です。プリフロップをレイズで終えたプレイヤーのハンドレンジにはキャップがありません。AdAcを持っている可能性があります。しかし、プリフロップをコールで止めたプレイヤーのハンドレンジにはキャップがあると推定できます。ポジションやベット回数によりますが、例えば6-maxのテーブルでUTGのオープンに対してコールしたBBのプレイヤーはAdAcを持ちえず、どんなに強くてもJdJcだと推定できるといった形です。

<Callout intent="note">
もちろん、トリッキーなプレイヤーはAdAcでコールに留めるかもしれません（実際のところ、SBからAdAcで一定頻度でリンプする均衡戦略もあります）。しかしここでの説明には影響しないため、便宜上AdAcは必ずレイズするものとして話を進めます。
</Callout>

## ハンドレンジを分類する観点

ハンドレンジを分類する時に用いる観点には、次のようなものがあります。ハンドレンジをこれらの観点と照らし合わせて考えることで、そのハンドレンジが置かれている状況を理解して分類できるようになります。

| 観点 | 意味 |
| --- | --- |
| **レンジキャップの有無** | ナッツハンドがレンジに含まれているかどうか |
| **バリューベットできるか** | EQやEQRが高いかどうか |
| **ショーダウンバリューの有無** | ワンペアのような何らかの役ができているなど、コールで勝っていることがあるかどうか |
| **アウツの有無と数** |（フロップまたはリバーにおいて）ストレートやフラッシュなどのドローがあるかどうか |

## ポラライズドレンジ

レンジキャップがなく、一定頻度のブラフを含むレンジを**ポラライズドレンジ（両極化レンジ)** と呼びます。言い換えると、ポラライズドレンジは**バリューレンジ**と**ブラフレンジ**が合わさったものです。

バリューレンジは**自信を持ってバリューベットできるハンドで構成されているハンドレンジ**で、レンジキャップがありません。 EQ と EQR の高いナッツハンドやそれに追随するハンドがバリューレンジになります。逆に、ブラフレンジは低EQのハンドで構成されているためブラフが主なポット獲得手段になるようなハンドレンジのことを指します。

ポラライズドレンジのアクションはミックスされている必要があります。バリューベットもブラフベットも同じ額で同じようにベットし、そのベットがバリューなのかブラフなのかがわからないようにします。違うのはどういう割合でバリューレンジとブラフレンジを混ぜ込むかです。

### ベット額とブラフ頻度の関係

ポラライズドレンジにおけるバリューレンジ部分とブラフレンジ部分はある一定の割合になるように分布されている必要があります。[均衡戦略](/game-theory-101#均衡戦略)となる割合よりもバリューの方が多いと、相手はコールする誘因がなくなり、無差別でなくなります。また、均衡割合よりもブラフの方が多いと今度はフォールドの誘因がなくなってしまいこれも無差別でなくなってしまいます。

ベット額をポットの $\frac{N}{M}$ としたとき、ブラフレンジのうち $\frac{N}{N+M}$ の頻度でベットすると均衡戦略となります（相手のコール/フォールドの選択を無差別にできます）。この均衡戦略での[EV](/glossary#ev)はレンジがキャップドでないぶんブラフによるEVが大きくなり、ベット額が高くなるに応じてEVも増加します（EV = $\frac{N+N+M}{(N+M)\times2}$）。

| ベット額 | 均衡戦略となる割合 | EV |
|:--- |:---:| ---:|
| ポットの $\frac14$ | バリュー 5 : ブラフ 1 | $\frac35$ Pot |
| ポットの $\frac13$ | バリュー 4 : ブラフ 1 | $\frac58$ Pot |
| ポットの $\frac12$ | バリュー 3 : ブラフ 1 | $\frac23$ Pot |
| ポットの $\frac23$ | バリュー 5 : ブラフ 2 | $\frac7{10}$ Pot |
| ポットの $\frac34$ | バリュー 7 : ブラフ 3 | $\frac57$ Pot |
| ポットと同額 | バリュー 2 : ブラフ 1 | $\frac34$ Pot |
| ポットの $\frac32$ | バリュー 5 : ブラフ 3 | $\frac45$ Pot |
| ポットの 2倍 | バリュー 3 : ブラフ 2 | $\frac56$ Pot |
| ポットの 10倍 | バリュー 11 : ブラフ 10 | $\frac{21}{22}$ Pot |

### ブラフレンジに向いたハンド

</Callout>

「$\frac14$ の頻度でブラフする」といった状況は、「どんなハンドでも4回に1回ブラフする」という意味ではありません。ブラフレンジ部分の中でどのハンドでブラフするかをきちんと選んでプレイした方がEVが高くなります。

なぜならブラフレンジにあるハンドの中には、僅かにショーダウンバリューがあるものもあります。例えばリバーでボードにカードが5枚開いた状態での強めのハイカードなどです。こういったハンドはほぼ間違いなく負けていますが、それでも「こちらのブラフレンジの上位が相手のコンデンスドレンジの下位と重複していて実際には勝っている」といった可能性を0にはできません。ショーダウンバリューは有るか無いかのバイナリではなく、大きなショーダウンバリューのあるハンドや僅かにしかショーダウンバリューのないハンドがあるようにグラデーションのような形で分布します。

ショーダウンバリューがあればあるほど、チェック/コールのアクションラインのEVがフォールドよりも高くなります。同時に、ブラフとして扱うよりもブラフキャッチャーとしての利用価値が高くなります。こうした理由から、ブラフに利用するのはショーダウンバリューの少ないレンジ下位のハンドが好ましくなります。

#### ブラフレンジの分布

また、EQの割にEVが高いハンドもブラフに回すべきです。リバー以降はベットアクションがないのでEQとEVが一致しますが、フロップやターンではアウツの多いドローハンドが往々にしてEQの割に大きなEVを保持しています。ドローハンドは前のストリートほどEVが高く、ターン、リバーと進むにつれてEQとEVが同じ値に収束していきます。また、ブラフが相手にコールされてしまってもアウツをドローできればバリューレンジのハンドとして返り咲くことができます。こうした理由から、前のストリートほどブラフによって得られるEVが大きくなり、ブラフ向きです。

## コンデンスドレンジ

（ポラライズドレンジと対比して）EQが中程度あることでショーダウンバリューがあるものの、バリューベットするにはEQが低くブラフベットとするにはEQが高すぎるようなマージナルハンドで構成されたハンドレンジを**コンデンスドレンジ（凝縮化レンジ)** と呼びます。凝縮化レンジはチェック/コールのアクションラインを取ることが多く、相手の消極的なアクションやフェイバリットなアウツのドローによって状況が変化するまでは積極的なベットアクションを取りません。コンデンスドレンジは相手のベットに対抗するシチュエーションが多く、ブラフキャッチのような難しい判断を迫られることになります。

### コンデンスドレンジでのコール頻度の関係

コンデンスドレンジができる主なアクションプランはブラフキャッチです。レンジキャップがあるぶんEVはポラライズドレンジよりも低くなり、一般的にマイナスEVです。

しかし、相手のベットに対して一定の割合でコールをしなければ相手はどんなハンドでもブラフベットできて際限なくEVを高めることができてしまいます。相手のベット額をポットの $\frac{N}{M}$ としたとき、 $\frac{M}{N+M}$ の頻度でコールすると均衡戦略になります（相手のブラフベットのEVを0にし、ブラフベットの有無を無差別にできます）。また、その時のEVは $\frac{M}{(N+M)\times2}$ で求められます。前述のとおり、常にポットの半分を下回るのでマイナスEVです。

| ベット額           | 均衡戦略となるコールの頻度 | EV                 |
| ---------------- | :------------------: | ------------------ |
| ポットの $\frac14$ | $\frac45$            | $\frac25$ Pot      |
| ポットの $\frac13$ | $\frac34$            | $\frac38$ Pot      |
| ポットの $\frac12$ | $\frac23$            | $\frac13$ Pot      |
| ポットの $\frac23$ | $\frac35$            | $\frac3{10}$ Pot   |
| ポットの $\frac34$ | $\frac47$            | $\frac27$ Pot      |
| ポットと同額        | $\frac12$            | $\frac14$ Pot      |
| ポットの $\frac32$ | $\frac25$            | $\frac15$ Pot      |
| ポットの 2倍       | $\frac13$      | $\frac16$ Pot       |
| ポットの 10倍      | $\frac{1}{11}$ | $\frac1{22}$ Pot |

### コール頻度から導き出すコール範囲

コール頻度に応じてどんなハンドでコールするかは、コンデンスドレンジの中でEVが上位X%のハンドをコールするようにすれば良いです。リバーではEQがそのままEVと一致しますが、フロップやターンなどのドローハンドが活きる場面ではボトムヒットのペアよりもフラッシュドローやストレートドローの方を優先した方が利益的になります。
